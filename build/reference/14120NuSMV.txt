<p>
NuSMV is a formal method notation that can be used check properties of behaviour models using logic.</p>

<p>The Umple NuSMV generator generates statements that can be used to analyse properties of state machine models.</p>

<p>More information about NuSMV can be found at the <a target="NuSMV" href="https://nusmv.fbk.eu">Alloy project home page, or on the <a target="NuSMVW" href="https://en.wikipedia.org/wiki/NuSMV">Wikipedia NuSMV page</a>.</p>

<p>This generator is discussed in detail in <a target="OpeyemiTh" href="https://dx.doi.org/10.20381/ruor-20549">Opeyemi Adesina's PhD thesis: Integrating Formal Methods with Model-Driven Engineering</a>.</p>
